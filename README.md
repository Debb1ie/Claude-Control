What I Love — Claude (HTML Website)

A visually rich personal webpage built using pure HTML and CSS.
This project focuses on typography, aesthetic layout, and interactive UI effects inspired by editorial-style websites.

The page highlights passions, storytelling sections, and a refined hero layout with animations and modern design techniques.

Overview

This website is a single-page design that showcases:

A cinematic hero section
Animated UI elements
Custom cursor effects
Scroll progress indicator
Passion cards
Deep storytelling layout
Timeline section
Smooth scrolling navigation

The design emphasizes elegant typography and warm color tones.

Technologies Used
HTML5
CSS3
Google Fonts
SVG background textures
CSS animations and transitions

Playfair Display

DM Mono
Features
1. Custom Cursor

The website includes a custom cursor system:

Cursor dot
Cursor ring animation
Hover interaction

Automatically disabled on touch devices.

2. Scroll Progress Bar

A progress bar at the top of the screen shows how far the user has scrolled through the page.

3. Animated Hero Section

The hero area includes:

Floating gradient orbs
Radial backgrounds
Animated text entrance
Scroll indicator
Statistics display
4. Navigation Bar

Sticky navigation with:

Blur effect when scrolling
Minimal typography design
Smooth scrolling behavior
5. Marquee Section

An animated scrolling text banner that:

Moves horizontally
Pauses on hover
Adds motion to the page
6. Passion Cards Grid

Interactive card system featuring:

Hover glow effect
Accent line animation
Soft shadow transitions
Icon containers

Responsive layout:

Desktop: 3 columns
Tablet: 2 columns
Mobile: 1 column
7. Deep Dive Section

Two-column editorial layout designed for storytelling content and quotes.

Includes:

Pull quotes
Long-form text formatting
Balanced typography
8. Timeline Layout

Vertical timeline used for:

Personal milestones
Journey highlights
Project history
